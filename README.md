# 项目简介

本项目是一款基于Python的车牌识别系统，包含源码及万字文档，旨在提供车牌号码和车牌颜色的识别功能。

## 核心功能

- 车牌号码识别
- 车牌颜色识别（绿牌、蓝牌、黄牌）

## 适用场景

- 交通监控
- 保安监控
- 车辆管理

## 技术特点

- 采用OpenCV进行图像处理
- 利用PyQt5构建用户界面
- 基于SVM进行车牌识别
